# node-gitlab-pages
基于node-gitlab api 和 web-hooks 所做，模仿github pages功能。

## 功能描述
1. 支持markdown语法及代码高亮。
2. 通过web请求pull-repo的master分支。
3. 发布到http://ip:8090/user/file-name。
